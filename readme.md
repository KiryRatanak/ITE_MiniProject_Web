## 📋 Project Overview

This blog project aims to document and preserve Khmer history, covering topics from ancient Angkorian times to modern Cambodia. It serves as an educational platform for students, history enthusiasts, and anyone interested in learning about Khmer heritage.

## 🚀 Features

- Blog posts about Khmer history topics
- Categories for different historical periods (Angkorian, Post-Angkorian, Modern)
- Responsive design for desktop and mobile viewing
- Image gallery of historical sites and artifacts
- Comment system for discussions
- Search functionality for blog posts
- Admin panel for creating/editing posts

## 📚 Topics Covered

- **Angkorian Era** (9th-15th centuries)
  - Angkor Wat and other temples
  - Great Khmer kings (Jayavarman II, Suryavarman II, Jayavarman VII)
  - Khmer architecture and art
  
- **Post-Angkorian Period**
  - Dark ages of Cambodia
  - French colonial period
  
- **Modern Cambodia**
  - Independence era
  - Cultural revival and preservation efforts
